核心能力

三层质量门禁 + 多 Agent 分工 + 可视化渲染

1

6 Agent 分工协作

DiscoveryAgent 选题 / ResearchAgent 取证 / PlanningAgent 规划 / WritingAgent 撰写 / VisualAgent 视觉 / ReviewerAgent 审核，各司其职

2

3 层 Reviewer 把关

evidence 模式审证据真实性 · content 模式审文案质量 · layout 模式审视觉布局，REVISE 自动返工

3

6 种视觉卡片

cover 封面 / timeline 时间线 / trend 趋势 / comparison 对比 / case 案例 / conclusion 结论，按 layout_spec 分发

4

Schema 校验下沉

可程序化软规则（字数/标签数/标题候选数）从 LLM Reviewer 下沉到 SchemaValidator，减少 token 消耗

5

证据可追溯

每个 external_claim / mixed 类型的 block 都必须有 claim_refs，引用 EvidencePack 中的 claim_id

6

Web 端可视化

FastAPI + Tailwind 实现 Linear 风格暗夜界面，14 态进度条 + iframe 预览 + 决策日志

本地运行

克隆仓库后按以下步骤启动

# 1. 安装依赖
pip install -r requirements.txt
pip install fastapi uvicorn

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 填入 SENSENOVA_API_KEY

# 3a. 启动 CLI 模式
python -m xhs_agent daily

# 3b. 启动 Web 模式（推荐）
cd src && python -m xhs_agent web
# → http://localhost:8000
